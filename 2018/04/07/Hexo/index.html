Hexo

  1. Node.js
  2. npm install -g hexo-cli
  3. hexo init BlogName
  4. cd BlogName
  5. npm install

Theme (以 Next 为例)

  1. cd BlogName
  2. git clone https://github.com/theme-next/hexo-theme-next themes/next

其他

  1. post.md

  2. hexo new page “categories”

  3. hexo new page “tags”

  4. hexo new page “about”

  5. <!-- more -->

  6. next/source/images/ 替换图标

Search (以 Algolia 为例)

  1. site->_config.yml

  2. export HEXO_ALGOLIA_INDEXING_KEY=Search-only API key

  3. hexo clean

  4. hexo algolia

  5. cd themes/next

  6. git clone https://github.com/theme-next/theme-next-algolia-instant-search source/lib/algolia-instant-search

  7. next/_config.yml

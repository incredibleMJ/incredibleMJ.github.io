Win10 + macOS Sierra(10.12.6 16G29) 双系统安装总结

发表于 2017-01-01 | 分类于 Other | 评论数：

Mac OS

安装方案确定

1. UEFI 还是 BIOS ? GPT 还是 MBR ? 

    - UEFI 和 BIOS 都是引导方式，BIOS 是较老的方式，推荐 UEFI

    - GPT 和 MBR 都是磁盘分区格式，MBR 是比较老的方式，推荐 GPT

2. Clover 还是 变色龙 ？

    - Clover 和 变色龙 都是 引导工具，用来启动 Mac OS 或 Windows

    - 变色龙配置较麻烦，对安装人员要求较高，安装效果还不见得比 Clover 好 

    - Clover 稍微简单点，可以通过一些配置解决大部分问题，推荐 Clover

3. 原版镜像还是懒人版 ？

    - 原版镜像对官方镜像修改较少，只添加一些必要驱动和破解补丁，自己按需加驱动

    - 懒人版就是加入大量驱动，做了较多修改，不用怎么改可能就安装成功了，但没有原版镜像安装稳定，推荐原版

    - 一般原版会比懒人版稳定，推荐原版

==总结：UEFI + GPT Clover + 原版镜像==

安装步骤概要

1. 制作启动U盘，有相应工具，这里不展开介绍

    * 注意 Clover 版本，尽量用较新版本
    * 注意 EFI/CLOVER/drivers64UEFI 文件夹下文件，这个与硬件相关
    * 注意 EFI/CLOVER/kexts 文件夹下文件，这些是自己加的驱动，硬件相关
    * 配置 EFI/CLOVER/config.plist ,硬件相关

    EFI 文件夹配置最简单的方式就是找一个与自己电脑配置差不多的成功案例，将他的 EFI 文件夹拿过来用，然后看报错信息做相应修改

    运气好的话，应该可以进安装界面了

2. 在安装界面找到磁盘工具，将分区或磁盘格式化为 HFS 格式

3. 选择新格式化好的用来安装 Mac OS 的分区，开始安装 (不是 GPT 分区的先重新分区，再格式化)

4. 进入系统后，用相应软件安装 Mac 版 Clover ，安装成功后将U盘里的 EFI 文件夹替换 Mac 盘下 EFI分区对应文件夹

经过上面几步应该可以初步使用了，下面的步骤就是优化阶段，驱动显卡 网卡 声卡等

==总结：启动U盘是关键==

后期优化

1. 显卡

    我的是 Nvidia 显卡，用的是官方的 web driver , 下载对应系统版本号的驱动就好了 (如遇到重启后黑屏，在 EFI/CLOVER/kexts/Other 文件夹下放入 lilu.kext NvidiaGraphicsFixup.kext NVWebDriverLibValFix.kext)

2. 网卡找自己网卡型号对应的驱动装上就好了

3. 声卡用的是万能驱动 VoodooHDA

4. 还有一些诸如 config.plist 优化 DSDT SSDT 电源管理 USB3.0 BIOS设置 等细节优化就不展开了，具体情况具体分析

==总结：显卡驱动是关键==

Windows 10

  1. Windows 10 官方出的U盘制作工具很给力，推荐装原版，进入系统后用 KMS 激活就好了

  2. 双硬盘的情况应先卸掉一张硬盘，保留一张主硬盘连接，因为 Windows 10 安装程序识别到有 EFI 分区了不会再新建了，如果 EFI 在副硬盘上，那么系统将从副硬盘引导，所以先只连接主硬盘，确保系统从主硬盘引导启动

  3. 遇到无法安装的情况，在安装界面 按下 shift + F10 打开命令行工具，利用 diskpart 重新分区 (注意格式化为 GPT , 创建 efi 、 msr 分区 以及你的其他分区)
